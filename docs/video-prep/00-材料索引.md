## 录制流程建议

```
准备阶段 (30min):
  1. 通读 01-脚本大纲 理解分段
  2. 打开 02-代码清单，在 IDE 中定位每个关键行
  3. 按 03-操作清单 完整走一遍演示
  4. 测试网络连接、确认 API 正常

正式录制:
  1. 翻到 01-脚本大纲 当前段落
  2. 对照 03-操作清单 执行操作
  3. 讲解时参考 02-代码清单 跳转代码
  4. 需要深度解释时翻查 实现原理详解.md

如果卡壳:
  - 暂停录制，看下一段脚本再重录这一段
  - 不必追求一遍过，片段拼接是正常的
```
