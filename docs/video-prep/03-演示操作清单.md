# 演示操作清单

> 录屏时一步一步照着点，防止卡壳。

---

## 演示 1：启动与自动加载

```
1. 双击运行 appqml1.exe
2. 等待 2-3 秒，观察：
   □ 应用窗口出现（960×640）
   □ 热门城市自动加载到仪表盘
   □ 动态天空背景开始渲染
   □ 暗色遮罩在白天自动变亮
3. 确认侧边栏 4 个导航按钮显示正常
```

---

## 演示 2：搜索城市

```
1. 点击仪表盘搜索框
2. 弹出 Popup，输入"北京"
3. 等待 300ms 防抖
4. 结果列表弹出
5. 键盘 ↓ 键选中第一个结果
6. 按 Enter 确认
   □ 焦点城市切换为"北京"
   □ 天气数据更新
   □ 天空背景根据天气码变化
   □ 侧边栏自动更新
```

---

## 演示 3：城市卡片交互

```
1. 搜索 3-4 个城市添加到列表
2. 单击某城市卡片
   □ 卡片绿色高亮
   □ 焦点切换为该城市
3. 双击某城市卡片
   □ 自动跳转到城市详情页
4. 鼠标悬停在卡片上
   □ 右上角出现关闭按钮 ×
   □ 卡片背景变亮
5. 点击关闭按钮
   □ 卡片从列表移除
```

---

## 演示 4：页面导航

```
1. 侧边栏点击"天气预报"
   □ 页面淡入淡出过渡（200ms）
   □ 预报卡片显示温度曲线
2. 点击范围选择器：24h → 72h → 7d
   □ 图表模式切换（hourly ↔ daily）
   □ 城市名 + 天气图标显示正常
3. 侧边栏点击"空气质量"
   □ 2×2 AQI 卡片网格
   □ 颜色圆点 + AQI 数字 + 污染物
4. 侧边栏点击"天文"
   □ GHI/DNI/DHI 数据卡片
   □ 日出日落时间 + 月相
5. 侧边栏点击"收藏"
   □ 收藏城市列表
   □ 图钉按钮固定到侧边栏
6. 侧边栏点击"设置"
   □ 切换暗色模式
   □ 调节最大卡片数
```

---

## 演示 5：城市详情页

```
1. 从仪表盘双击某城市卡片
   □ 进入城市详情页
2. 逐项检查数据卡片：
   □ 实时天气（温度、体感、风向、湿度等）
   □ 空气质量（AQI、污染物）
   □ 3 日预报（日期 + 图标 + 温度范围）
   □ 24 小时预报（水平滚动列表）
   □ 生活指数（4 列网格）
   □ 日出日落 + 月相
   □ 天气预警（如果有）
3. 点击星标收藏/取消收藏
```

---

## 演示 6：调试面板（重点）

```
1. 按 Ctrl+Shift+B
   □ 调试面板出现在右上角
2. 点击"Debug"模式
3. 拖动"时间"滑块
   □ 天空从白天渐变到黄昏到夜晚
   □ 观察"太阳高度"数值变化
   □ 星星在夜晚出现
4. 在"天气码模拟"中输入"302"
   □ 点击"应用"
   □ 云层激活、雨层出现、闪电闪烁
5. 点击 Layer 状态查看
   □ Cloud active=true, tp=1.0
   □ Weather active=true, tp=1.0
6. 拖动"云 tp"滑块到 0.5
   □ 云层半透明渐变
7. 取消勾选"淡入"
   □ 图层切换变为即时
8. 点击"预设"ComboBox → 选择"104-阴"
   □ 云覆盖率增加到 100%
```

---

## 演示 7：自动演示模式

```
1. 在调试面板中点击"▶ 演示"
   □ 面板变透明
   □ 天空从凌晨 0 点开始自动变化
   □ 时间持续前进
   □ 天气码循环切换（晴→多云→阴→雨→雪→雾...）
2. 勾选/取消"快速"切换速度
3. 点击"■ 停止"结束演示
```

## 演示 8：着色器效果（侧重点）

```
1. 打开调试面板，切换到 Debug 模式
2. 把时间调到 12:00（正午）
   □ 天空正午蓝色，有太阳光晕
3. 调到 6:00（日出）
   □ 金色时刻，地平线暖色调
4. 调到 21:00（夜晚）
   □ 深蓝天空，星星闪烁
5. 天气码设 300（阵雨）
   □ 雨丝倾斜下落
6. 天气码设 400（小雪）
   □ 雪花飘落
7. 天气码设 500（雾）
   □ 地平线雾浓度更高
8. 天气码设 302（雷阵雨）
   □ 闪电间歇闪烁
```

## 演示 9：配置持久化

```
1. 在调试面板中调好参数
2. 输入天气码"300"
3. 点击"保存配置"
4. 关闭应用，重新启动
5. 查看天气码 300
   □ 恢复保存的渲染参数
```

## 演示 10：缓存与离线

```
1. 首次打开应用（已联网）
   □ 数据加载正常
```

## 快捷键速查

| 按键 | 功能 |
|------|------|
| `Ctrl+Shift+B` | 打开/关闭调试面板 |
| `↑/↓` | 搜索框中键盘导航 |
| `Enter` | 确认搜索结果 |
| `双击城市卡片` | 跳转城市详情 |
| `单击城市卡片` | 切换焦点城市 |

## 录屏检查项

- [ ] 窗口尺寸固定为 960×640，无缩放变形
- [ ] 天气 API 有网络连接（演示前先测试）
- [ ] 鼠标光标可见性设置开启
- [ ] 系统音量/麦克风电平静音或关闭
- [ ] 关闭无关弹窗和通知
- [ ] 代码字体 ≥ 14px，录屏时清晰可读
- [ ] IDE 侧边栏折叠，只展示源码区域
- [ ] 录制分辨率 1920×1080 或 2560×1440
