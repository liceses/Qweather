## 一、架构决策：C++ 计算 + QML 纯渲染

### 1.1 与项目现有模式的统一

```
现有模式:                        V3 新增:
───────────────────              ────────────────
C++ Store → 信号 → QML 绑定     C++ BackgroundManager → SkyState → QML Layer 绑定
C++ API 通过 contextProperty     C++ GlobalClock 通过 contextProperty
QML 不解析 JSON，纯绑定           QML 不算天文/天气码，纯绑定
```

项目已建立 "C++ 预处理 → QML 纯渲染" 范式。V3 不引入新范式，只追加新模块。

### 1.2 为什么不选 V2.2

| V2.2 (QML Manager) | V3 (C++ Manager) |
|---|---|
| QML 中计算 sunProgress + 天空色 | C++ `AstronomyModel` 中计算 |
| QML 中 `typeFromCode()` 映射 | C++ `WeatherProfile::fromCode()` O(1) 查表 |
| 6 个独立背景文件，"NightRain"需特殊处理 | Layer 栈组合，"夜间+雨"=SkyLayer(夜)+RainLayer |
| QML JS 过渡编排防抖 | C++ `TransitionController` 编排放 `QTimer::singleShot` |
| 增加天气效果=新 shader 文件 | 增加天气效果=改 `WeatherProfile` 映射表 |

**选择 V3 的理由**：V3 在第一次写对架构之后，后续所有变更（新天气效果、新 Layer、性能升级）几乎不需要改 QML 绑定——只需改 C++ 映射表或追加一个 Layer shader。

## 二、Layer 栈架构

```
ApplicationWindow
├── WeatherBackground (QML Item)           ← Layer 栈容器 + 视差
│   ├── z= 0  SkyLayer                    ← 始终渲染: 天空渐变+太阳+月亮+星星
│   ├── z=10  AtmosphereLayer             ← 始终渲染: twilight/散射/曝光/地平线色调
│   ├── z=20  Loader { CloudLayer }       ← 条件: cloudCoverage>0.01
│   ├── z=30  Loader { WeatherLayer }     ← 条件: rainIntensity+snowIntensity>0.001
│   ├── z=40  Loader { FogLayer }         ← 条件: fogDensity>0.001
│   └── z=50  Loader { LightningLayer }   ← 条件: lightningProb>0
├── [现有 UI: Rectangle 底色 + RowLayout + StackLayout] ← 从 Main.qml 的 Rectangle 渐变升级
└── WeatherBackgroundDebugPanel           ← 浮动面板 z=100
```

### 2.1 Layer 渲染规则

| Layer | 始终渲染 | 条件渲染 | GPU 负载 | 备注 |
|---|---|---|---|---|
| SkyLayer | ✅ | — | 天空渐变 + 1 sun + 1 moon + 50 stars | 全时段基础层 |
| AtmosphereLayer | ✅ | — | 色调后处理 pass | 轻量 |
| CloudLayer | — | ✅ | 2~3 层 fBM noise | 晴空时卸载 |
| WeatherLayer | — | ✅ | rain 或 snow 粒子 ≤60 | 天气时激活 |
| FogLayer | — | ✅ | 色调叠加 | 雾/霾/沙尘时激活 |
| LightningLayer | — | ✅ | 间歇全屏闪 | 雷暴时激活 |

**最坏情况 GPU 负载**：雷阵雨+雾+云层→4 个条件 Layer 同时激活，总粒子循环 ~60，远在安全范围内。

### 4.3 GlobalClock.h

```cpp
#pragma once
#include <QObject>
#include <QTimer>

class GlobalClock : public QObject {
    Q_OBJECT
    Q_PROPERTY(double elapsed READ elapsed NOTIFY tick)
public:
    explicit GlobalClock(QObject *parent = nullptr);
    double elapsed() const { return m_elapsed; }
signals:
    void tick();
private slots:
    void onTimer();
private:
    QTimer m_timer;
    double m_elapsed = 0.0;
};
```

### 4.4 GlobalClock.cpp

```cpp
#include "GlobalClock.h"

GlobalClock::GlobalClock(QObject *parent) : QObject(parent) {
    m_timer.setInterval(16); // ~60 FPS
    connect(&m_timer, &QTimer::timeout, this, &GlobalClock::onTimer);
    m_timer.start();
}

void GlobalClock::onTimer() {
    // 应用非活跃时 QTimer 自然暂停，恢复后继续累加
    m_elapsed += 0.016;
    emit tick();
}
```

#pragma once
#include <QObject>

class AstronomyModel : public QObject {
    Q_OBJECT
    Q_PROPERTY(float sunProgress   READ sunProgress   NOTIFY updated)
    Q_PROPERTY(float solarAltitude READ solarAltitude NOTIFY updated)
    Q_PROPERTY(float solarAzimuth  READ solarAzimuth  NOTIFY updated)
    Q_PROPERTY(float moonAltitude  READ moonAltitude  NOTIFY updated)
    Q_PROPERTY(float moonAzimuth   READ moonAzimuth   NOTIFY updated)
    Q_PROPERTY(float moonPhase     READ moonPhase     NOTIFY updated)
    Q_PROPERTY(float moonIllum     READ moonIllum     NOTIFY updated)

public:
    explicit AstronomyModel(QObject *parent = nullptr);

    float sunProgress()   const { return m_sunProgress; }
    float solarAltitude() const { return m_solarAltitude; }
    float solarAzimuth()  const { return m_solarAzimuth; }
    float moonAltitude()  const { return m_moonAltitude; }
    float moonAzimuth()   const { return m_moonAzimuth; }
    float moonPhase()     const { return m_moonPhase; }
    float moonIllum()     const { return m_moonIllum; }

    void setSunTimes(int sunriseMinutes, int sunsetMinutes);
    void setMoonData(int phaseIcon, float illumination);
    Q_INVOKABLE void update(qint64 nowMsecs = -1);

    // skyColorsForProgress: 7时段×3色混合，供 BackgroundManager 调用
    struct SkyColors {
        float r_top, g_top, b_top;
        float r_horiz, g_horiz, b_horiz;
        float r_amb, g_amb, b_amb;
    };
    SkyColors colorsForSunProgress(float sp) const;

    bool isNight() const { return m_sunProgress < -0.05f || m_sunProgress > 1.05f; }

signals:
    void updated();

private:
    static constexpr float PI = 3.14159265359f;
    float m_sunProgress    = 0.5f;
    float m_solarAltitude  = 45.0f;
    float m_solarAzimuth   = 180.0f;
    float m_moonAltitude   = 30.0f;
    float m_moonAzimuth    = 90.0f;
    float m_moonPhase      = 4.0f;
    float m_moonIllum      = 1.0f;
    int   m_sunriseMin     = 360;   // 06:00 默认
    int   m_sunsetMin      = 1080;  // 18:00 默认
    float m_lat            = 39.9f; // 北京默认纬度

    float calcSunAltitude(float sp) const;
    float calcSunAzimuth(float sp) const;
};

#### AstronomyModel.cpp 关键实现

```cpp
#include "AstronomyModel.h"
#include <QDateTime>
#include <cmath>

AstronomyModel::AstronomyModel(QObject *parent) : QObject(parent) {}

void AstronomyModel::setSunTimes(int sunriseMin, int sunsetMin) {
    m_sunriseMin = sunriseMin;
    m_sunsetMin  = sunsetMin;
    update();
}

void AstronomyModel::setMoonData(int phaseIcon, float illumination) {
    // 和风天气 moonPhase icon: 800=新月 ... 804=满月 ... 807=残月
    // 映射为 0.0(新月) ~ 0.5(满月) ~ 1.0(残月→下一个新月)
    m_moonPhase = (phaseIcon - 800) / 8.0f;
    m_moonIllum = illumination;
    update();
}

void AstronomyModel::update(qint64 nowMsecs) {
    if (nowMsecs < 0)
        nowMsecs = QDateTime::currentMSecsSinceEpoch();
    int nowMin = static_cast<int>((nowMsecs / 60000) % 1440); // 0~1439

    int dayLen = m_sunsetMin - m_sunriseMin;
    if (dayLen <= 0) dayLen = 720; // fallback 12h
    m_sunProgress = float(nowMin - m_sunriseMin) / float(dayLen);

    // 太阳高度: 简化的正弦模型
    float sp = m_sunProgress;
    m_solarAltitude = calcSunAltitude(sp);
    m_solarAzimuth  = calcSunAzimuth(sp);

    // 月亮位置: 与太阳对跖
    float moonSP = std::fmod(sp + 0.5f, 1.0f);
    m_moonAltitude = calcSunAltitude(moonSP);
    m_moonAzimuth  = calcSunAzimuth(moonSP);

    emit updated();
}

float AstronomyModel::calcSunAltitude(float sp) const {
    // 简化: sin(π*sp) 产生拱形, 加入纬度因子
    float latRad = m_lat * PI / 180.0f;
    float maxAlt = 90.0f - std::abs(m_lat);
    return std::sin(PI * sp) * maxAlt;
}

float AstronomyModel::calcSunAzimuth(float sp) const {
    // 日出东方(90°)→正午正南(180°)→日落西方(270°)
    return 90.0f + sp * 180.0f;
}

AstronomyModel::SkyColors AstronomyModel::colorsForSunProgress(float sp) const {
    // 7 时段: < -0.3 深夜, -0.3~0 蓝调, 0~0.15 金粉,
    //          0.15~0.7 白天, 0.7~0.9 暖午后, 0.9~1.0 橙红, >1.0 紫调
    SkyColors c;
    // 采用 mix 在相邻时段间平滑插值 (简化展示核心结构)
    if (sp < -0.3f) {          c = {0.04f,0.04f,0.10f, 0.05f,0.05f,0.16f, 0.02f,0.02f,0.06f}; }
    else if (sp < 0.0f) {      c = {0.10f,0.16f,0.35f, 0.29f,0.19f,0.38f, 0.04f,0.04f,0.13f}; }
    else if (sp < 0.15f) {     c = {0.29f,0.44f,0.65f, 0.96f,0.64f,0.38f, 0.88f,0.44f,0.31f}; }
    else if (sp < 0.7f) {      c = {0.29f,0.56f,0.85f, 0.53f,0.81f,0.92f, 0.78f,0.88f,0.94f}; }
    else if (sp < 0.9f) {      c = {0.35f,0.54f,0.71f, 0.83f,0.60f,0.42f, 0.75f,0.69f,0.63f}; }
    else if (sp < 1.0f) {      c = {0.23f,0.35f,0.54f, 0.88f,0.47f,0.25f, 0.63f,0.19f,0.19f}; }
    else if (sp < 1.3f) {      c = {0.16f,0.10f,0.29f, 0.38f,0.13f,0.25f, 0.04f,0.04f,0.13f}; }
    else {                      c = {0.04f,0.04f,0.10f, 0.05f,0.05f,0.16f, 0.02f,0.02f,0.06f}; }
    return c;
}
```

struct WeatherProfile {

class WeatherProfileDB {

    static float snowIntensity(int code);

WeatherProfile WeatherProfileDB::fromCode(int code, bool isDay) const {
    // 夜间统一查 m_nightProfiles (优先)，回退 m_dayProfiles
    if (!isDay && m_nightProfiles.contains(code))
        return m_nightProfiles[code];
    if (m_dayProfiles.contains(code))
        return m_dayProfiles[code];
    return WeatherProfile{}; // 默认晴
}

### 8.4 TransitionController.cpp 关键实现

```cpp
#include "TransitionController.h"
#include <QTimer>
#include <cmath>

TransitionController::TransitionController(QObject *parent) : QObject(parent) {}

void TransitionController::onSkyStateChanged(const SkyState &s) {
    ++m_transitionId;
    SkyState prev = m_prev;
    m_prev = s;

    // === Cloud ===
    bool cloudNow = s.cloudCoverage > 0.01f;
    bool cloudWas = prev.cloudCoverage > 0.01f;
    if (cloudNow && !cloudWas) {
        animateIn(m_cloudTP, m_cloudActive, 100);
    } else if (!cloudNow && cloudWas) {
        animateOut(m_cloudTP, m_cloudActive, 600);
    }

    // === Weather ===
    bool weatherNow = (s.rainIntensity + s.snowIntensity) > 0.001f;
    bool weatherWas = (prev.rainIntensity + prev.snowIntensity) > 0.001f;
    if (weatherNow && !weatherWas) {
        int delay = cloudNow ? 400 : 100;
        animateIn(m_weatherTP, m_weatherActive, delay);
    } else if (!weatherNow && weatherWas) {
        animateOut(m_weatherTP, m_weatherActive, 600);
    }

    // === Fog ===
    bool fogNow = s.fogDensity > 0.001f;
    bool fogWas = prev.fogDensity > 0.001f;
    if (fogNow && !fogWas) {
        animateIn(m_fogTP, m_fogActive, 100);
    } else if (!fogNow && fogWas) {
        animateOut(m_fogTP, m_fogActive, 400);
    }

    // === Lightning ===
    m_lightningActive = s.lightningProb > 0.0f;
    emit lightningActiveChanged();
}

void TransitionController::animateIn(float &tp, bool &active, int delayMs) {
    int localId = m_transitionId;
    QTimer::singleShot(delayMs, this, [this, &tp, &active, localId]() {
        if (m_transitionId != localId) return;
        active = true;
        tp = 1.0f;
        emit /* activeChanged */;
        emit /* tpChanged */;
    });
}

void TransitionController::animateOut(float &tp, bool &active, int delayMs) {
    tp = 0.0f;
    emit /* tpChanged */;
    int localId = m_transitionId;
    QTimer::singleShot(delayMs, this, [this, &active, localId]() {
        if (m_transitionId != localId) return;
        active = false;
        emit /* activeChanged */;
    });
}
```

// shaders/common/noise.glsl
// 2D 噪声 + fBM，被 CloudLayer 等引用

float hash21(vec2 p) {
    float h = dot(p, vec2(127.1, 311.7));
    return fract(sin(h) * 43758.5453);
}

float noise2D(vec2 p) {
    vec2 i = floor(p);
    vec2 f = fract(p);
    f = f * f * (3.0 - 2.0 * f);
    return mix(mix(hash21(i), hash21(i+vec2(1,0)), f.x),
               mix(hash21(i+vec2(0,1)), hash21(i+vec2(1,1)), f.x), f.y);
}

float fbm(vec2 p, int octaves) {
    float v = 0.0, a = 0.5; vec2 s = p;
    for (int i = 0; i < 5; i++) {
        if (i >= octaves) break;
        v += a * noise2D(s);
        s *= 2.0; a *= 0.5;
    }
    return v;
}

### 9.2 common/particle.glsl（示例）

```glsl
// shaders/common/particle.glsl
// 雨滴/雪花/冰雹复用函数

float rainDrop(vec2 uv, float id, float t, float wind, float intensity) {
    float seed = fract(sin(id * 731.491) * 43758.5453);
    float x = mod(seed * 6.28318 + sin(t * 0.8 + seed * 6.28) * 0.03 * wind, 1.0);
    float speed = mix(0.3, 0.9, seed);
    float y = fract(seed * 3.14159 + t * speed);
    vec2 pos = vec2(x, y);
    float dist = abs(uv.x - pos.x) * 200.0;
    float vdist = abs(uv.y - pos.y);
    float tail = smoothstep(0.0, 0.03, vdist) * 0.7;
    float body = 1.0 - smoothstep(0.0, 0.002, dist);
    return max(body * tail, 0.0);
}

float snowFlake(vec2 uv, float id, float t, float wind, float intensity) {
    float seed = fract(sin(id * 127.1 + dot(vec2(id, 0.0), vec2(311.7, 269.5))) * 43758.5453);
    float sway = sin(t * 0.8 + seed * 6.28) * 0.03 * wind;
    float x = mod(seed + sway, 1.0);
    float speed = mix(0.08, 0.25, seed);
    float y = fract(seed * 6.28 + t * speed);
    vec2 pos = vec2(x, y);
    float d = distance(uv, pos);
    float size = mix(0.004, 0.014, fract(seed * 983.7));
    return (1.0 - smoothstep(0.0, size, d)) * smoothstep(size, size * 0.5, d);
}
```

### 9.4 shaders/cloud.frag（关键：transitionProgress 驱动云漂入）

```glsl
#version 440
layout(location=0) in vec2 qt_TexCoord0;
layout(location=0) out vec4 fragColor;

layout(std140, binding=0) uniform buf {
    mat4 qt_Matrix;
    float qt_Opacity;
    float time;
    float cloudCoverage;
    float windSpeed;
    int   variant;
    float transitionProgress;
} ubuf;

// fbm 定义见 common/noise.glsl

void main() {
    float tp = clamp(ubuf.transitionProgress, 0.0, 1.0);
    vec2 uv = qt_TexCoord0;

    // tp=0: 云在屏幕右侧外(+0.6), tp=1: 正常位置
    float offset = mix(0.6, 0.0, tp);
    float coverage = mix(0.0, ubuf.cloudCoverage, tp);

    // 云层1
    float speed1 = ubuf.windSpeed * 0.03 + 0.01;
    float cloud1 = fbm(vec2((uv.x - 0.3 + offset + ubuf.time * speed1) * 3.0, uv.y * 4.0), 3);
    // 云层2 (反方向)
    float cloud2 = fbm(vec2((uv.x - 0.3 - offset * 0.7 + ubuf.time * speed1 * 0.7) * 3.5, uv.y * 3.5), 2);
    float cloud = max(cloud1, cloud2 * 0.6);

    float cloudMask = smoothstep(1.0 - coverage, 1.0 - coverage + 0.15, cloud);
    vec3 sky = vec3(0.53, 0.81, 0.92); // 透明天空
    vec3 cloudColor = vec3(0.82, 0.84, 0.87);
    vec3 color = mix(sky, cloudColor, cloudMask);
    fragColor = vec4(color, cloudMask * 0.9) * ubuf.qt_Opacity;
}
```

### 9.5 shaders/rain.frag（关键：粒子循环上限固定）

```glsl
#version 440
layout(location=0) in vec2 qt_TexCoord0;
layout(location=0) out vec4 fragColor;

layout(std140, binding=0) uniform buf {
    mat4 qt_Matrix;
    float qt_Opacity;
    float time;
    float intensity;
    float windSpeed;
    int   variant;       // 0=普通雨, 1=雷暴, 2=冰雹, 3=雷暴+冰雹
    float transitionProgress;
    int   particleLimit; // 60/40/20
} ubuf;

const int MAX_DROPS = 60;
// rainDrop, hailDrop 定义见 common/particle.glsl

void main() {
    float tp = clamp(ubuf.transitionProgress, 0.0, 1.0);
    // tp<0.001 整帧跳过 (性能优化)
    float activeIntensity = ubuf.intensity * tp;
    if (activeIntensity < 0.001) {
        // 输出纯色背景
        fragColor = vec4(0.35, 0.38, 0.42, 1.0) * ubuf.qt_Opacity;
        return;
    }
    vec2 uv = qt_TexCoord0;
    int activeDrops = int(mix(0.0, float(ubuf.particleLimit), activeIntensity));
    float rain = 0.0;
    for (int i = 0; i < MAX_DROPS; i++) {
        if (i >= activeDrops) break;
        rain += rainDrop(uv, float(i) + floor(ubuf.time * 0.5), ubuf.time, ubuf.windSpeed, activeIntensity);
    }
    rain = clamp(rain, 0.0, 1.0);
    vec3 bg = vec3(0.35, 0.38, 0.42);
    vec3 color = mix(bg, vec3(0.75, 0.8, 0.85), rain);
    fragColor = vec4(color, 1.0) * ubuf.qt_Opacity;
}
```

## 十五、GPU 性能预算

> 每个 Layer 单帧 GPU 时间上限。禁止在已上线的 shader 中无限堆效果。

| Layer | 目标 (1080p) | 说明 |
|---|---|---|
| SkyLayer | < 0.5 ms | 天空渐变 + 1 太阳 + 1 月亮 + 50 星星 |
| AtmosphereLayer | < 0.3 ms | 色调后处理单次 pass |
| CloudLayer | < 1.0 ms | 2~3 层 fBM (octaves 3/2/2) |
| RainLayer / SnowLayer | < 1.0 ms | 粒子循环 ≤60，tp=0 整帧跳过 |
| FogLayer | < 0.5 ms | 色调叠加，无粒子 |
| LightningLayer | < 0.2 ms | 间歇全屏闪（仅雷暴时激活） |
| **总计（最坏情况 4 条件 Layer 同时激活）** | **< 3.0 ms** | @60fps = 16.7ms/帧，GPU 预算 ~18% |
| **总计（通常情况 0~2 条件 Layer）** | **< 1.5 ms** | 晴天仅 Sky + Atmosphere，GPU 预算 ~9% |

**测量方式**：Qt 6 内置 `QSG_RENDER_TIMING=1` 环境变量输出每帧 GPU 时间。每个 Layer 的 shader 复杂度由以下因素控制：

| 因素 | 上限 | 控制方式 |
|---|---|---|
| 粒子循环上限 | 60 (high) / 40 (mid) / 20 (low) | `particleLimit` uniform (编译时常量 `MAX_DROPS=60`) |
| fBM octaves | 5 (high) / 3 (mid) / 2 (low) | `AppSettings.perfLevel` → QML dynamic property |
| 星星数量 | 50 (high) / 35 (mid) / 20 (low) | 同 particleLimit 模式 |
| tp=0 跳过 | 整帧 return | `if (activeIntensity < 0.001) return;` |

## 十六、风险与缓解

| 风险 | 概率 | 影响 | 缓解 |
|------|:---:|:---:|------|
| `qt_add_shaders` 在 Windows/MinGW 下不支持某些 GLSL 版本 | 中 | P0 | 先用 `#version 440`；失败则降为 `300 es` 再试 |
| 7 个 Layer Loader z-stack 导致首次加载卡顿 | 低 | P1 | 条件 Layer 默认 `active: false`，首次不加载 |
| `SkyState` Q_GADGET 在 QML 中绑定更新频率过高 | 低 | P2 | Manager 60s 一次 `emit skyStateChanged`；parallax 独立信号 |
| `TransitionController` `QTimer::singleShot` 中 lambda 捕获引用在对象析构后悬空 | 低 | P0 | `m_transitionId` 检查 + 对象生命周期由 parent-child 保证 |
| 大量新增文件导致 CMake 初次配置时间过长 | 低 | P2 | 单次可接受 (< 5s) |
